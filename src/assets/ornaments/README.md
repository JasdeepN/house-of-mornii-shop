# Ornamental Assets

This directory contains SVG ornamental designs inspired by regal Art Nouveau and heritage opulence, crafted specifically for House of Mornii.

## Available Assets

### 1. Corner Flourish (`corner-flourish.svg`)
An intricate corner ornament featuring:
- Double-line corner frames
- Diamond-shaped accents
- Decorative curves and flourishes
- Accent dots for visual interest

**Usage**: Perfect for card corners, section borders, and decorative frames.

### 2. Divider Ornament (`divider-ornament.svg`)
An elegant horizontal divider with:
- Gradient fade-out lines
- Central diamond and circle motif
- Symmetrical side flourishes
- Small decorative elements

**Usage**: Section separators, content dividers, visual breaks between elements.

### 3. Header Ornament (`header-ornament.svg`)
A vertical ornamental element featuring:
- Central diamond with nested circle
- Top and bottom flourishes
- Bilateral symmetry
- Delicate decorative curves

**Usage**: Navigation accents, header decorations, vertical separators.

## Color Palette

All ornaments use the House of Mornii darker gold color:
- Primary Gold: `oklch(0.60 0.11 78)` - A refined, muted antique gold
- Variations use opacity for depth and visual hierarchy

## Technical Details

- **Format**: SVG (Scalable Vector Graphics)
- **Styling**: Inline CSS classes for easy customization
- **Responsiveness**: Fully scalable without quality loss
- **Browser Support**: All modern browsers

## Customization

To modify colors, update the CSS custom properties in the `<style>` block:

```css
.ornament-gold {
  stroke: oklch(0.60 0.11 78); /* Change this value */
}
.ornament-fill {
  fill: oklch(0.60 0.11 78); /* Change this value */
}
```
